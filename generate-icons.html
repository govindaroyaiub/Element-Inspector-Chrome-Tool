Element Inspector - Icon Generator

This page generates the required PNG icons for the Chrome extension.

16x16


32x32


48x48


128x128


Instructions

  1. Click "Generate All Icons" to create the icon graphics
  2. Right-click each icon and select "Save image as..." or use the download buttons
  3. Save them as:
    • icon16.png (16x16)
    • icon32.png (32x32)
    • icon48.png (48x48)
    • icon128.png (128x128)
  4. Place all PNG files in the "icons" folder
  5. The Chrome extension will now have proper icons
